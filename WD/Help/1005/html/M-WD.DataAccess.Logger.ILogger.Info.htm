Remarks

This method first checks if this logger is INFO enabled by comparing the level of this logger with the log4net.Core.Level.Info level. If this logger is INFO enabled, then it converts the message object (passed as parameter) to a string by invoking the appropriate log4net.ObjectRenderer.IObjectRenderer. It then proceeds to call all the registered appenders in this logger and also higher in the hierarchy depending on the value of the additivity flag. WARNING Note that passing an System.Exception to this method will print the name of the System.Exception but no stack trace. To print a stack trace use the Info(object,Exception) form instead. Shahid Kochak, Asim, 7/21/2017.
